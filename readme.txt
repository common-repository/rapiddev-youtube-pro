== Description ==

Advanced YouTube features. Add live subscriptions to your website, post livechat from broadcasts, embed videos and more!

This plugin is not an official plugin created by YouTube. The YouTube name is owned by Google INC. and is used to facilitate identification.

== Installation ==

Method using FTP
1. Upload `rapiddev-youtube` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Method using Plugins Manager
1. Go to Dashboard > Plugins > Add new
2. Select the option to upload to the server
3. Choose the file rapiddev-youtube.zip


== Frequently asked questions ==

= What features does this plugin have? =

With the help of this plugin you will add widgets to your website that can display the latest live broadcast or the latest movie.
You can also add a widget - subscribe now.
In addition, you'll see statistics on the channel you chose in the administrative panel 

= How do I go to settings? =

In the plug-in manager next to Creator Tools for YouTube the 'Settings' button will be displayed.
You can find the plug-in settings under the 'YouTube Pro' tab in the 'Settings' section of your cockpit

= Does the plugin support many languages? =

This plugin contains a full translation into Polish and English.

= Where can I get the API V3 key for my statistics? =

You can create your codes on this page:
https://console.developers.google.com/apis/credentials

= Does the plug have any security features? =

The plugin has integrated protection against outdated versions of WordPress and PHP. It also detects the lack of consistency of plugin files.

= Do the creators of the plugin collect any information about me? =

No, by adding Maintenance Pro to your website, you also create your own reCaptcha key (but you do not have to). The maintenance mode only works on your site.
